GeneValidator

Tool to identify problematic gene predictions based on comparison with gene predictions in public databases. Funded by NESCent Google Summer of Code 2013.

You can run GeneValidator locally from the command line or from a web browser (suitable for <100 queries). Results can be viewed in a browser in either case.

  • Instructions to install and run GeneValidator from command line.
  • Instructions to install and run GeneValidator from a web browser.

You can use the online web app hosted by our lab for small (< 10) queries to try out GeneValidator. See example output for a few randomly selected transcript and protein sequences across various species: transcripts | proteins
