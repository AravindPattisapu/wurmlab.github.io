Anurag Priyam Research Assistant

I am interested in improving the state of software tooling available to Biologists. Software I have created include: an easy to use BLAST server - SequenceServer, virtual environments using docker - oswitch, platform to crowd-source manual curation - Afra and annotation liftover pipeline - flo. Software I have helped to create include: identification of problems with gene prediction - GeneValidator and modular bioinformatics library for NodeJS - bionode.
