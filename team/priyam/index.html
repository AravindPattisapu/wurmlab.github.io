Anurag Priyam Research Assistant

I develop software, create analysis pipelines and run research support tasks such as gene prediction and providing sequence search and genome browser server for Biologists.

Analysis of 'omics data interests me.

LinkedIn   |   Twitter   |   GitHub   |   Scholar

Software

  • SequenceServer - simple to setup, easy to use BLAST server.
  • oswitch - virtual environments for Bioinformatics analysis.
  • GeneValidator - identify problems with gene prediction.
  • Afra - platform to crowd-source manual curation.
  • flo - annotation liftover pipeline.
Contributions
  • GenomeTools - versatile open source genome analysis software.
  • bionode - modular bioinformatics library for NodeJS.
  • JBrowse - genome browser.
