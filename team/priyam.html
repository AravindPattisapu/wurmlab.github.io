Anurag Priyam

BBSRC-funded creator of a tool for crowd-sourcing curation of gene predictions. Created the easy BLASTing tool - Sequenceserver. Created the tool to facilitate running of analyses and creating pipelines using docker. Expert in Ruby and JavaScript. [GitHub].

Software

  • SequenceServer: BLAST search made easy!
  • Afra: Genome annotations for the masses!
  • oswitch: One-line access to other operating systems!
  • flo: Same species annotation lift over pipeline
